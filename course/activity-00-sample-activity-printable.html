    4.

    In this problem, state the definitions of the following properties of binary relations: Reflexive, Anti-reflexive or Irreflexive, Symmetric, Antisymmetric, and Transitive.
    ðŸ”—
    Solution.
    For all of these, let \(R\) be a binary relation on a set \(A\text{.}\)
    Reflexive:
    ðŸ”—
    \(R\) is reflexive if and only if for every \(x \in A\text{,}\) \(x R x\text{.}\)
    ðŸ”—
    Anti-Reflexive or Irreflexive:
    ðŸ”—
    \(R\) is anti-reflexive or irreflexive if and only if for every \(x \in A\text{,}\) \(x \not \mathrel{R} x\) (i.e., \(x\) is not related to itself).
    ðŸ”—
    Symmetric:
    ðŸ”—
    \(R\) is symmetric if and only if for every all \(x,y \in A\text{,}\) \(x R y\) if and only if \(y R x\text{.}\) (Note that this also means that \(R\) is symmetric if and only if for all \(x,y \in A\text{,}\) \(x \not \mathrel{R} y\) if and only if \(y \not \mathrel{R} x\text{.}\))
    ðŸ”—
    Antisymmetric:
    ðŸ”—
    \(R\) is antisymmetric if and only if for all \(x,y \in A\text{,}\) if \(x \ne y\text{,}\) then \(x R y\) and \(y R x\) cannot both be true. (So, for every pair of distinct elements in \(A\text{,}\) one of the following holds: \(xRy\) but \(y \not \mathrel{R} x\text{,}\) \(y R x\) but \(x \not \mathrel{R} y\text{,}\) or \(x \not \mathrel{R} y\) and \(y \not \mathrel{R} x\text{.}\))
    ðŸ”—
    Transitive
    ðŸ”—
    \(R\) is transitive if and only if for all \(x,y,z \in A\text{,}\) if \(x R y\) and \(y R z\text{,}\) then \(x R z\text{.}\) (Note that these donâ€™t necessarily have to be distinct.)

    5.

    For each of the following problems, consider the given relation \(R\) defined on a set \(A\text{.}\) Determine whether \(R\) satisfies any of the properties defined in Problem 4 above. If a property does not hold, say why.
    ðŸ”—

    (a)

    \(R = \{(a,a), (b,b), (c,c), (d,d), (a,b), (b,a)\}\) on the set \(A = \{a,b,c,d\}\text{.}\)
    ðŸ”—
    Solution.
    The relation is reflexive, symmetric, and transitive. Clearly it is not irreflexive. Moreover, \(R\) is not antisymmetric because there is a distinct pair of elements \(a,b\) where \(a R b\) and \(b R a\) are both true.
    ðŸ”—
    ðŸ”—
    ðŸ”—

    (b)

    Consider the relation defined on \(A = \Z\) as \(x R y\) if \(|x-y|\lt1\text{.}\)
    ðŸ”—
    Solution.
    Since \(x,y \in \Z\text{,}\) \(xRy \iff |x-y|\lt1 \iff x=y\text{.}\) Then this relation is clearly going to be reflexive, symmetric, and transitive.
    ðŸ”—
    ðŸ”—
    ðŸ”—

    (c)

    Consider the relation defined on \(A = \Z\) where we declare \(x R y\) if and only if \(x\) and \(y\) have the same parity.
    ðŸ”—
    Solution.
    This relation is reflexive because clearly \(x\) will have the same parity as itself, so \(x R x\text{.}\) Also, \(R\) is symmetric because if \(xRy\text{,}\) so \(x\) and \(y\) have the same parity, then \(y\) and \(x\) will also have the same parity, so \(yRx\text{.}\) The relation is transitive because if \(xRy\) and \(yRz\text{,}\) then \(x\) and \(y\) have the same parity and \(y\) and \(z\) have the same parity, so \(x\) and \(z\) thus have the same parity. Hence, \(x R z\text{.}\) (Notice that \(R\) is actually just congruence modulo 2.)
